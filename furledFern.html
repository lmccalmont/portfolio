The Furled Fern

This project is a e-Commerce site for a mock plant shop. It was created for a solo, semester-long project. I used HTML, CSS, and vanilla JavaScript. The checkout system is fully usable, which was created using Paypal Sandbox. I also created the SVG logo myself using Inkscape. The homepage contains featured plants, upcoming events, and site navigation. The checkout/online shop page has a filtering system and shopping basket with total price.
